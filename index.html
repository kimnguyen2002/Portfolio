Symptom Checker with Drug Info - Streamlit App


Symptom Checker is a basic web-based tool developed using Streamlit that allows users to quickly predict minor illnesses based on their selected symptoms. By utilizing a Decision Tree Classifier, this application provides an initial diagnosis, display related drug options for the predicted condition, sourced from a drug side effects dataset.

View the app Source code repository

Retrieving Biological Data


My focus is to query the Europe PMC API to retrieve information related to the term "gene". By constructing the API request with the specified query and parameters, the code fetches the relevant data in JSON format and stores it in the "gen_data" variable for further use or analysis.

The key purpose of this code is to programmatically access and extract data from the EBI Europe PMC API based on a specific search query, allowing the user to incorporate this data into their broader application or research workflow.

View the source code

White Wine Classification - Streamlit App


This Streamlit App is used to automatically predict different types of white wine based on three Machine Learning models: Random Forest, Support Vector Machine and Logistic Regression.
