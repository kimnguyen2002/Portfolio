Anemia Diagnostic Assistant - Streamlit App



This project presents a web application designed to predict the risk of anemia using advanced machine learning techniques. By inputting their hematological parameters, users can receive an accurate anemia diagnosis based on a predictive model that has been rigorously trained. Inspired by the pioneering research of Mojumdar et al. (2025), this project is a seamless integration of clinical expertise and cutting-edge data science. It marks a significant leap forward in transforming the way we diagnose and intervene in anemia, enabling early detection and better patient care. The success of this project is the result of an exceptional collaboration: Dr. Gem Wu from Chang Gung Memorial Hospital provided invaluable medical insights, while my dedicated data science partner played a crucial role in developing and refining the model.

Heart Risk Prediction - Streamlit App



The Heart Risk Prediction is a web application designed to assess the risk of heart disease based on user-input health metrics. Utilizing machine learning models, the app provides a user-friendly interface through Streamlit, offers insights into the user's heart health, categorizing their risk as low or high. Symptom Checker with Drug Info - Streamlit App



Symptom Checker is a basic web-based tool developed using Streamlit that allows users to quickly predict minor illnesses based on their selected symptoms. By utilizing a Decision Tree Classifier, this application provides an initial diagnosis, displays related drug options for the predicted condition, sourced from a drug side effects dataset.

Retrieving Biological Data


My focus is to query the Europe PMC API to retrieve information related to the term "gene". By constructing the API request with the specified query and parameters, the code fetches the relevant data in JSON format and stores it in the "gen_data" variable for further use or analysis.

The key purpose of this code is to programmatically access and extract data from the EBI Europe PMC API based on a specific search query, allowing the user to incorporate this data into their broader application or research workflow.
